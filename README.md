# 7-Segment-LED-Blinking-Using-16-Bit-Timer-Mode-on-AT89C51-Microcontroller
This project demonstrates blinking a 7-segment LED display using the 16-bit Timer Mode (Mode 1) of the AT89C51 (8051) microcontroller with Embedded C. The timer generates accurate delays to control the blinking operation. The project is developed using Keil uVision and simulated in Proteus.
